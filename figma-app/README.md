
  # ddd

  This is a code bundle for ddd. The original project is available at https://www.figma.com/design/TlfFyxIIuO53gUtRHGCV85/ddd.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Repository structure note

  This repository contains two separate apps:

  - `figma-app/` (active app to develop and modify)
  - `old-app/` (legacy app kept as a separate project)

  Keep the two apps isolated: do not merge their `index.html` files and do not move files between folders unless explicitly requested.
